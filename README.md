# Simple Twitter clone

A simple twitter clone that can create new posts and register/authorize a user with a JWT Token.

#### Технологии

* Spring 
  
  * Spring Boot
  
  * Spring Data Jpa
  
  * Spring Security
  
  * Starter Web

* React JS

* DataBase - PostgreSQL
